想要创造一门专属于你的编程语言吗？ 亦或是... 你早已对各类编程语言的设计和实现感到好奇？

这本书，为你而准备。

《Crafting Interpreters》 包含实现一门功能完备且高效的脚本语言所需的一切知识。 除了掌握词法分析与语法分析等基本概念，你还会接触到字节码表示与垃圾回收等等底层细节。 你的大脑将充满新的想法，你的双手将码出厚厚老茧。这是一场理论与实践的狂欢！

从众人皆知的 main() 函数开始，你将逐渐地构建出一种具有丰富语法、 动态类型、垃圾回收、词法作用域、头等函数、闭包、类与继承的编程语言。 所有内容都将包含在几千行干净整洁的代码中，不必担心无法理解这些东西， 因为它们将全部产自你手。

关于 Robert Nystrom

许多年以前，利用陪产期间夜里给孩子喂奶的闲暇之余，我开始钻研编程语言 Bug, 逐渐沉迷其中，无法自拔。 在全职做编程语言领域的工作之前，我还鼓捣出了许多 业 余 作 品 。如今我正在 Google 工作，研究 Dart 语言。

在我爱上编程语言之前，我在 Electronic Arts 开发了八年的游戏。根据在那里学到的知识， 我写了一本名为《Game Programming Patterns》的畅销书。这本书也是可以完全免费阅读的。
